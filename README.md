Clone the project from below repository:
https://github.com/siva1711/bdd-selenium-interview.git

Run the below command to build the project
mvn clean install

Once the build is successful, run the tests using,
RunnerTest.java

Project is developed with windows chrome driver exe file (refer to /test/resources/driver/chromedriver.exe),
please update this file when running in mac or different chrome version

TestData.properties file is used to store the gloal test parameters like browsername and driver path.
